## Описание проекта

Веб-приложение для автоматического анализа фотографий чеков и учёта личных расходов.  
Позволяет:
- распознавать текст с чеков (в будущем — OCR)
- категоризировать траты по правилам и графу знаний
- вести диалог с чат-ботом для запросов о связях между магазинами, категориями и товарами

## Выполненные лабораторные работы

| № | Название | Описание | Основные технологии |
|---|----------|----------|---------------------|
| 1 | Базовый интерфейс | Hello World + проверка установки | Streamlit |
| 2 | Продукционная модель | Rule-Based логика + правила в JSON | JSON, if-else, pandas |
| 3 | Объектная модель + Граф знаний | Классы объектов и связи (NetworkX) | dataclasses, NetworkX, matplotlib |
| 4 | Диалоговый интерфейс (Chatbot v1.0) | Чат-бот с историей сообщений и поиском в графе | Streamlit chat, session_state |
