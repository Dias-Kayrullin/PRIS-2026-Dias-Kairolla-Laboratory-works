# PRIS-2026: Анализ расходов по чекам

Проект по дисциплине «Проектирование и разработка интеллектуальных систем»  

Автор: Dias Kairolla

## Описание проекта

Веб-приложение для учёта и анализа личных расходов по чекам.  
Основные возможности:
- визуализация связей между магазинами, категориями и товарами (граф знаний)
- диалоговый интерфейс (чат-бот), который отвечает по заранее заданным правилам
- добавление и удаление расходов через чат
- просмотр истории трат в виде таблицы и графика

Чат-бот работает **чисто по правилам** (if-else + поиск в графе), без внешних ИИ-моделей и без Ollama — чтобы любой мог запустить без дополнительных установок.
